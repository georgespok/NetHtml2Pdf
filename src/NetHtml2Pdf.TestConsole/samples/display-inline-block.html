CSS Display: Inline-Block - Test Fixture

This fixture demonstrates inline-block element behavior. Inline-block elements should:
  • Flow inline (side-by-side) when space allows
  • Wrap as whole units when space runs out
  • Support width, height, margins, padding, and borders
  • Align vertically to middle by default
  • Participate in inline flow as atomic boxes
